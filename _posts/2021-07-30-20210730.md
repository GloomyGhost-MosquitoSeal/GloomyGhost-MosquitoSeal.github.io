### 修改网络验证服务器

```bash
nano /etc/NetworkManager/conf.d/20-connectivity.conf
```

将网址`uri`替换

```bash
[connectivity]
uri=http://nmcheck.gnome.org/check_network_status.txt
```

### 强制关闭网络验证

```bash
[connectivity]
enabled=false
```
