![image](/assets/post/20211214/5.jpg)

## 点灯点灯

先看看PD22是哪一个IO

```
cat /sys/kernel/debug/pinctrl/2000000.pinctrl/pinmux-pins
```
![image](/assets/post/20211214/6.jpg)

嗯，是118脚，导出这个脚

```
echo 118 > /sys/class/gpio/export
```

点灯

```
cd /sys/class/gpio/gpio118
echo out>direction
echo 1 > value  #LED熄灭
echo 0 > value  #LED点亮
```

然后亮了，顿时索然无味
